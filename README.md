# 🏛️ AIMTIA — أمتيا

**الشريك الاستراتيجي الوطني لتصميم وإدارة وتنفيذ الفعاليات والمؤتمرات والمبادرات الوطنية**

A premium enterprise-grade website designed for **AIMTIA**, showcasing national-scale events, conferences, exhibitions, and strategic initiatives aligned with Saudi Vision 2030.

---

## 📋 نظرة عامة | Overview

أمتيا هي منصة رقمية مؤسسية حديثة تم تصميمها لتعكس مكانة الشركة كشريك وطني استراتيجي للجهات الحكومية والهيئات والشركات الكبرى في المملكة العربية السعودية.

يقدم الموقع تجربة رقمية فاخرة تجمع بين:

* الهوية المؤسسية الراقية
* سرد قصصي بصري (Storytelling)
* عرض المشاريع والإنجازات الوطنية
* إبراز الأثر الاقتصادي والاجتماعي
* مواءمة مباشرة مع رؤية المملكة 2030

---

## ✨ أبرز المميزات | Key Features

### 🎬 Hero Cinematic Experience

واجهة افتتاحية سينمائية مع صور متحركة وتأثيرات بصرية احترافية تعكس حجم المشاريع الوطنية.

### 📊 National Impact Dashboard

عرض مؤشرات الأداء والأثر الوطني من خلال أرقام وإحصائيات تفاعلية.

### 🏛️ Enterprise Authority Layer

إبراز الشراكات مع الجهات الحكومية والهيئات الوطنية لتعزيز الموثوقية المؤسسية.

### 🗺️ Nationwide Coverage

عرض التغطية الجغرافية للمشاريع والفعاليات داخل مختلف مناطق المملكة.

### 🎯 Vision 2030 Integration

ربط مباشر بين خدمات الشركة ومستهدفات رؤية المملكة 2030.

### 📁 Interactive Projects Showcase

معرض مشاريع احترافي يعرض المؤتمرات والمعارض والفعاليات المنفذة.

### 📰 Insights & Knowledge Hub

قسم للمقالات والتحليلات والأبحاث المتخصصة في قطاع الفعاليات والمؤتمرات.

### 📱 Fully Responsive

تصميم متجاوب يعمل بسلاسة على:

* الهواتف الذكية
* الأجهزة اللوحية
* الحواسيب المكتبية
* الشاشات الكبيرة

### ⚡ High Performance

أداء سريع وتجربة مستخدم سلسة مع تحسينات متقدمة للواجهة.

* متصفح حديث
* لا توجد مكتبات أو أطر عمل مطلوبة

### التثبيت

#### 1. استنساخ المشروع

```bash
git clone https://github.com/your-username/AIMTIA.git
```

#### 2. الانتقال للمجلد

```bash
cd AIMTIA
```

#### 3. تشغيل الموقع

افتح ملف:

```bash

```

أو استخدم:

```bash

```
